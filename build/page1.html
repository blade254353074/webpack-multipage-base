CommonJs: require.ensure

The `require.ensure` method ensures that every dependency in `dependencies` can be synchronously required when calling the `callback`. `callback` is called with the `require` function as parameter.
Note: `require.ensure` only loads the modules, it doesn’t evaluate them.

CommonJs require.ensure compoents/movies.js
Callback executed!
compoents/movies.js loaded!

从 Console 中可以看出：
CommonJs 的 require.ensure 会先读取模块（并不执行，是根据 ensure 来确认依赖的），
在 Callback 中执行 require(module) 后，模块才执行。
即体现出，CommonJs 的同步性：没有网络 IO 延迟（实际上已经跟随入口文件一起加载了），require(module) 即执行模块。
